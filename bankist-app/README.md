# [Bankist App](https://bankistapp-rodrigo.netlify.app/)

Bankist App is a fictional banking application created during my JavaScript learning journey. It simulates basic banking operations such as logging in, viewing transactions, transferring money, requesting loans, closing an account, sorting account movements, formatting dates, and handling a logout timer.

This project was mainly built to practice JavaScript array methods, numbers, dates, timers, and dynamic UI updates.

## Features

- User login with username and PIN
- Display of deposits and withdrawals
- Balance, income, outcome, and interest calculation
- Transfers between accounts
- Loan request validation
- Account closing
- Transaction sorting
- Date, currency, and timer handling

## Technologies used

- CSS

- Parcel

```bash

```

# [Bankist App](https://bankistapp-rodrigo.netlify.app/)

Ela simula operações bancárias básicas, como login, visualização de transações, transferência de dinheiro, solicitação de empréstimo, encerramento de conta, ordenação de movimentações, formatação de datas e controle de um timer de logout.

Este projeto foi desenvolvido principalmente para praticar métodos de arrays em JavaScript, números, datas, timers e atualizações dinâmicas da interface.

- Manipulação de datas, moeda e timer
