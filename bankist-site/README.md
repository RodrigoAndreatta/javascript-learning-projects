# [Bankist Site](https://bankistsite-rodrigo.netlify.app/)

Bankist Site is a fictional landing page created during my JavaScript learning journey. It was built to practice more advanced DOM manipulation and browser event handling techniques.

This project helped me understand how modern websites can feel more interactive by using smooth scrolling, event delegation, tabbed components, sticky navigation, section reveal animations, lazy loading images, and a custom slider.

## Features

- Modal window
- Smooth scrolling
- Event delegation for page navigation
- Tabbed component
- Menu fade animation
- Sticky navigation
- Section reveal on scroll
- Lazy loading images
- Slider with buttons, keyboard controls, and dots

## Technologies used

- HTML
- CSS
- JavaScript
- Parcel

## How to run locally

```bash
npm install
npm run start
```

## Build command

```bash
npm run build
```

For Netlify, use `npm run build` as the build command and `dist` as the publish directory.

# [Bankist Site](https://bankistsite-rodrigo.netlify.app/)

Bankist Site é uma landing page fictícia criada durante a minha jornada de aprendizado em JavaScript. Ela foi desenvolvida para praticar técnicas mais avançadas de manipulação do DOM e manipulação de eventos do navegador.

Este projeto me ajudou a entender como sites modernos podem se tornar mais interativos usando smooth scrolling, event delegation, componentes em abas, navegação sticky, animações de revelação de seções, lazy loading de imagens e um slider personalizado.

## Funcionalidades

- Janela modal
- Smooth scrolling
- Event delegation para navegação da página
- Componente em abas
- Animação de fade no menu
- Navegação sticky
- Revelação de seções ao rolar a página
- Lazy loading de imagens
- Slider com botões, controles pelo teclado e dots

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Parcel

## Como rodar localmente

```bash
npm install
npm run start
```

## Comando de build

```bash
npm run build
```

Para o Netlify, use `npm run build` como comando de build e `dist` como diretório de publicação.
