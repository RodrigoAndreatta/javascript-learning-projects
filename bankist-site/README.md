# Bankist Site

Bankist Site is a fictional landing page created during my JavaScript learning journey. It was built to practice more advanced DOM manipulation and browser event handling techniques.

This project helped me understand how modern websites can feel more interactive by using smooth scrolling, event delegation, tabbed components, sticky navigation, section reveal animations, lazy loading images, and a custom slider.

## Features

- Modal window
- Smooth scrolling
- Event delegation for page navigation
- Tabbed component
- Menu fade animation
- Sticky navigation
- Section reveal on scroll
- Lazy loading images
- Slider with buttons, keyboard controls, and dots

## Technologies used

- HTML
- CSS
- JavaScript
- Parcel

## How to run locally

```bash
npm install
npm run start
```

## Build command

```bash
npm run build
```

For Netlify, use `npm run build` as the build command and `dist` as the publish directory.
