# [Pig Game](https://piggame-rodrigo.netlify.app/)

Pig Game is a two-player dice game built after learning more about DOM manipulation, events, functions, and application state. Compared to the first projects, this one required more control over game flow and UI state.

This project helped me practice organizing repeated logic and managing a more interactive user interface.

## Features

- Two-player turn system
- Dice roll generation
- Current and total score tracking
- Hold and reset actions
- Winner state styling

## Technologies used

- HTML
- CSS
- JavaScript
- Parcel

## How to run locally

```bash
npm install
npm run start
```

## Build command

```bash
npm run build
```

For Netlify, use `npm run build` as the build command and `dist` as the publish directory.
